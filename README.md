# ALSO Cloud Events

GitHub Pages eventkalender en agenda-abonnement voor Cloud partners van ALSO Nederland B.V.

## Beheer

Je beheert events in één bestand:

```text
events.yml
```

Na iedere wijziging draait GitHub Actions automatisch en worden de website, Event Builder, JSON-feed en agenda opnieuw opgebouwd.

## GitHub Pages

Gebruik:

```text
Source: Deploy from a branch
Branch: main
Folder: /docs
```
