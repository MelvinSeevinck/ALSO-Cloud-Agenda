Outlook Desktop: handmatig abonneren

Outlook Desktop kan vanuit een website niet betrouwbaar automatisch naar het juiste invoerveld springen. De knop hierboven kopieert daarom de webcal-link. Plak die vervolgens bij Abonneren via web.

  1. Open Outlook en ga naar Agenda.
  2. Kies Agenda toevoegen.
  3. Kies links Abonneren via web.
  4. Plak de gekopieerde link in het bovenste veld.
  5. Geef de agenda de naam ALSO Cloud Agenda.
  6. Kies eventueel een kleur en klik op Importeren.
